# @nnviz/web — React frontend

Vite + React app. **Deploy this folder to Vercel.**

## Vercel settings

| Setting | Value |
|---------|--------|
| Root Directory | `apps/web` |
| Output Directory | `dist` |
| Framework | Vite |

See [../../VERCEL.md](../../VERCEL.md) for full deployment guide.

## Local dev

From the repo root:

```bash
pnpm dev
```

Or from this folder (after `pnpm install` at repo root):

```bash
pnpm dev
```
